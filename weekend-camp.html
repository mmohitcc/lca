Weekend Coding Camps

Our Weekend coding camps are designed to build core STEM and technology skills in a group setting. Kids with the same interest and skill level are grouped together to achieve a common goal under an instructor's guidance. Our camps teach Computer Science Fundamentals using Scratch, Python, Javascript. Kids learn building websites with HTML, CSS. Our Robotics program is based on building and programming robots.
